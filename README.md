# librespeed-template


A BastilleBSD Template for [librespeed](https://github.com/librespeed/speedtest)


## Bootstrap
```
bastille bootstrap https://github.com/ddowse/librespeed-template
```

## Apply
```
bastille template TARGET ddowse/librespeed-template
```
